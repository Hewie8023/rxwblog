# 介绍

这是在今年秋招的时候复习C++整理的基础知识点，可能不全面。C++的知识点太多太复杂，这里只是一些常见的问题和知识点，如果要深入学习，推荐以下几本书：

  • 《C++ Primer》
  • 《Effective C++》
  • 《深度探索C++对象模型》
  • 《STL源码剖析》

希望对之后的学习和面试稍有帮助。

如果您觉得有意思，欢迎鼓励一个Star
